# Data Dictionary: scapis_simulated.csv

Simulated SCAPIS-like dataset with 1,000 observations and 53 variables.
Generated by `scripts/00_generate_data.R` with `set.seed(42)`.

## Identifiers

| Variable | Type | Description |
|----------|------|-------------|
| `id` | character | Participant ID (SCAPIS_0001 to SCAPIS_1000) |

## Demographics

| Variable | Type | Range / Levels | Description |
|----------|------|----------------|-------------|
| `age` | numeric | 50.0–64.0 | Age in years (uniform distribution) |
| `sex` | character | Male, Female | Biological sex (49% male, 51% female) |

## Anthropometry

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `bmi` | numeric | >= 16 | Body mass index (kg/m^2); mean ~27, SD ~4.3. **3% MCAR missingness.** |
| `waist_cm` | numeric | >= 55 | Waist circumference (cm); sex-dependent means (M: 98, F: 88) |

## Blood Pressure

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `sbp` | numeric | >= 85 | Systolic blood pressure (mmHg); mean ~130, SD ~18 |
| `dbp` | numeric | >= 50 | Diastolic blood pressure (mmHg); mean ~80, SD ~10 |

## Lipids

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `total_chol` | numeric | >= 2.5 | Total cholesterol (mmol/L); mean ~5.5, SD ~1.0 |
| `hdl` | numeric | >= 0.5 | HDL cholesterol (mmol/L); sex-dependent (M: 1.3, F: 1.6). **MAR missingness** (~10% in males, ~4% in females). |
| `ldl` | numeric | >= 0.5 | LDL cholesterol (mmol/L); Friedewald-derived: total_chol - hdl - triglycerides/2.2 |
| `triglycerides` | numeric | >= 0.3 | Triglycerides (mmol/L); log-normal distribution, median ~1.4. **5% MCAR missingness.** |

## Glucose Metabolism

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `fasting_glucose` | numeric | >= 3.0 | Fasting plasma glucose (mmol/L); mean ~5.6, SD ~0.9. **4% MCAR missingness.** |
| `hba1c` | numeric | >= 20 | Glycated hemoglobin (mmol/mol); mean ~38, SD ~6. **MAR missingness** (~10% if age > 60, ~4% otherwise). |

## Inflammation

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `hs_crp` | numeric | >= 0.1 | High-sensitivity C-reactive protein (mg/L); log-normal, median ~1.5. **MAR missingness** (~14% in current smokers, ~6% otherwise). |

## Renal Function

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `creatinine` | numeric | >= 35 | Serum creatinine (umol/L); sex-dependent (M: 88, F: 70) |

## Lifestyle

| Variable | Type | Levels | Description |
|----------|------|--------|-------------|
| `smoking` | character | Never (45%), Former (35%), Current (20%) | Smoking status |
| `physical_activity` | character | Low (25%), Moderate (50%), High (25%) | Self-reported physical activity level |

## Medications

| Variable | Type | Levels | Description |
|----------|------|--------|-------------|
| `antihypertensive` | integer | 0, 1 | Antihypertensive medication use (30% prevalence) |
| `statin` | integer | 0, 1 | Statin use (20% prevalence) |
| `diabetes_med` | integer | 0, 1 | Diabetes medication use (8% prevalence) |

## Pulmonary Function

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `fev1_pct` | numeric | >= 30 | FEV1 % predicted; mean ~95, SD ~14 |
| `fvc_pct` | numeric | >= 35 | FVC % predicted; mean ~98, SD ~13 |

## Coronary Artery Calcium Score (CACS)

| Variable | Type | Range / Levels | Description |
|----------|------|----------------|-------------|
| `cacs` | numeric | >= 0 | Continuous CACS (Agatston units); ~58% zero, log-normal among positives |
| `cacs_binary` | integer | 0, 1 | CACS > 0 indicator (~42% positive) |
| `cacs_cat` | character | 0, 1-100, 101-400, >400 | Categorical CACS (clinical risk groups) |

CACS is generated from a latent propensity model driven by age, sex, smoking, SBP, total cholesterol, and fasting glucose.

## Proteins (25 Olink NPX)

## Survival Data

| Variable | Type | Range / Levels | Description |
|----------|------|----------------|-------------|
| `time_to_event` | numeric | 0–6 | Time to CVD event or censoring (years); median ~4.5 |
| `cvd_event` | integer | 0, 1 | CVD event indicator (~8% event rate) |
| `event_type` | character | MI, Stroke, NA | Event subtype (MI ~60%, Stroke ~40%); NA if censored |

Survival times are generated from a Weibull model with hazard driven by age, sex, CACS category, smoking, and GDF15.

## Correlation Structure

- The 10 CACS-associated proteins share a block correlation of r = 0.20.
- Null proteins 11–15 (RETN through FABP4) share a mild correlation of r = 0.10.
- Remaining null proteins are independent of each other.
